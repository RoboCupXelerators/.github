## 中文

### 机器人足球，作为完整系统来构建

Xelerator 是一支面向 RoboCup Junior Soccer 的机器人足球队伍，围绕机械结构、视觉感知、边缘 AI、LiDAR 定位、仿真训练和比赛策略构建完整的自主机器人系统。

我们做工程时始终围绕一个现场问题：这项工作能不能让机器人在真实比赛里更稳定、更容易调试、更可靠地完成动作？

### 我们在做什么

| 方向 | 重点 |
| --- | --- |
| 视觉与感知 | 多摄像头感知、YOLO 检测、场线、球、对手、障碍和复杂光照适配。 |
| 机械与执行 | 盘带器、击球器、避震、换电、传感器视野，以及能承受赛场碰撞的结构。 |
| 策略软件 | 统一场地坐标、底盘接口、传感器哨兵值和可复用的攻防状态机。 |
| 仿真训练 | 基于 Unity 的真值、UDP 控制、相机流和多机器人联调验证。 |

### 最新机器：Terminus

Terminus 是 Xelerator 面向 2026 赛季的新一代机器人，在检测、无刷执行、自定位、模块化装配和电气架构上完成升级，用于应对复杂光照、碰撞和比赛现场维护。

当前公开资料中的关键提升：

- 检测误差下降 97%
- 预计处理速度提升 400%
- 检测准确率提升 90%
- 执行能耗下降 50%
- 无刷执行性能提升 70%
- 装配时间下降 63%

查看详情：[Terminus 新机页面](https://xelerator.argonserver.net/machine.html)

### 工程闭环

比赛、装机、调参和复盘是同一个系统工程。队伍持续迭代原型、车端控制、仿真、视觉部署、定位、现场 SOP 和真实赛况回归，直到机器人能在高压比赛环境中稳定工作。

主要工具与技术方向：

- YOLO / 模型部署
- 四摄像头感知与标定
- LiDAR 场地定位
- RDK X5 与边缘 AI 加速
- 3D 建模与模块化机械设计
- Unity 仿真与 UDP 控制

## English

### Robot Soccer, Built As A Complete System

Xelerator is a RoboCup Junior Soccer robotics team building autonomous robot soccer systems across mechanical design, visual perception, edge AI, LiDAR positioning, simulation, and match strategy.

Our work is centered on one field-tested question: does this make the robot more stable, faster to debug, and more reliable during real competition?

### What We Build

| Area | Focus |
| --- | --- |
| Vision and perception | Multi-camera sensing, YOLO detection, field lines, ball tracking, opponents, obstacles, and lighting adaptation. |
| Mechanical execution | Dribblers, kickers, suspension, battery swaps, sensor visibility, and structures that can survive field collisions. |
| Strategy software | Unified field coordinates, chassis interfaces, sensor sentinel values, and reusable attack-defense state machines. |
| Simulation training | Unity-based validation with ground truth, UDP control, camera streams, and multi-robot integration tests. |

### Latest Machine: Terminus

Terminus is our latest robot generation for the 2026 season. It upgrades detection, brushless actuation, self-positioning, modular assembly, and the electrical architecture for complex lighting, collisions, and on-site maintenance.

Highlights from the current public material:

- 97% lower detection error
- 400% expected processing speed gain
- 90% detection accuracy gain
- 50% lower actuation energy use
- 70% brushless actuation gain
- 63% shorter assembly time

Read more on the [Terminus machine page](https://xelerator.argonserver.net/machine.html).

### Engineering Loop

Competition, assembly, tuning, and review are treated as one engineering loop. The team iterates through prototypes, onboard control, simulation, vision deployment, localization, field SOPs, and real-match regression until the robot can perform under pressure.

Core tools and directions include:

- YOLO / model deployment
- Four-camera perception and calibration
- LiDAR-based field positioning
- RDK X5 and edge AI acceleration
- 3D modeling and modular mechanical design
- Unity simulation and UDP control
